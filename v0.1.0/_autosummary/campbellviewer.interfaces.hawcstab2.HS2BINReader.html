class campbellviewer.interfaces.hawcstab2.HS2BINReader(file, parent=None)#

Bases: object

Extract mode shapes out of HAWCStab2 binary files.

The HAWCStab2 model is set up hierarchically in this way:

TURBINE
    |
    --> substructure i
            |
            |
            --> bodies
            |   |
            |   --> body ii
            |   |        |
            |   |        --> body data s (arc position for each element)
            |   |
            |   --> body ii + 1
            |       |
            |       --> body data s (arc position for each element, 1d array)
            |
            --> opstate jj (states, dictionary)
                |
                --> bodies (list)
                    |
                    --> body jjj
                        |
                        --> mode jjjj
                        |   |
                        |   --> ua0 (2d array)
                        |
                        --> mode jjjj + 1
                            |
                            --> ua0 (2d array)

Matlab struct: self.substructure[i_subs].opstate[istate].body_result(ibody).mode(imode).ua0

Parameters:

file (str) –

substructure#

list of substructures present in a turbine

Type:

list

num_modes#

number of modes stored in the binary file

Type:

int

num_steps#

number of state points e.g. wind steps/ rpm steps

Type:

int

operational_data#

(N,3) map of operational data i = operation point, j = 0 -> wind speed, j = 1 -> pitch, j = 2 -> power

Type:

ndarry

__offset#

current offset of bytes in bin file, position in file/cached string

Type:

int

__buff#

cached binary file

Type:

bytes

__num_DOF#

parameter for the number of DOFs = u_x, u_y, u_z, theta_x, theta_y, theta_z

Type:
