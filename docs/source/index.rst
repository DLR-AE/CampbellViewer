CampbellViewer
=======================================
Unified plotting tool for Campbell diagrams, processing results of wind turbine
simulation tools HAWCStab2 and Bladed. The main functionality of the tool is to
interactively visualize Campbell diagrams.

Functionalities in a nutshell
------------------------------

- Interactively plot Campbell diagrams of different tools, e.g., for comparison
- Plot participation factors of modes
- Load results from database
- Save results to database
- Export plot to PDF
